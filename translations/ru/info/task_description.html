Список запретных слов:

Входные данные: Массив, как список (list) целых чисел (int).

Выходные данные: Сумма чисел, как целое число (int).

Примеры: 

checkio([1, 2, 3]) == 6
checkio([2, 2, 2, 2, 2, 2]) == 12
