Our new calculator has censorship and it does not accept some words. You should trick this thing and write the program for calculation the sum of numbers.

Given a list of numbers, you should find sum of numbers. Your solution should not contain any of forbidden words (even as part of another word).

The list of forbidden words:

  • sum
  • import
  • for
  • while
  • reduce

Input: A list of numbers.

Output: The sum of numbers.

Example: 

checkio([1, 2, 3]) == 6
checkio([2, 2, 2, 2, 2, 2]) == 12
